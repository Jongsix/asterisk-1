asterisk.cr - Asterisk PBX all-in-one library
=============================================

Asterisk PBX libraries (ARI, AMI and all types of AGI) for crystal-lang.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  asterisk:
    github: ctiapps/asterisk
```
